JODO 特性

JODO 是一个轻量级命令行 Todo 应用程序，帮助您高效管理日常任务。

主要特性

  • 简洁的命令行界面
  • 支持任务截止日期设置
  • 重要任务标记
  • 截止日期彩色显示（逾期、紧急、即将到期）
  • 批量添加和操作任务
  • 多语言支持（中文、英文、日语）
Terminal
$ jodo --help
JODO - 简单的命令行 Todo 应用
用法: jodo [选项] [任务描述]
选项:
-l, --list 列出所有任务
-c, --complete 完成任务
-d, --delete 删除任务
-t, --time 设置截止日期
-h, --help 显示帮助信息
-v, --version 显示版本信息
-m, --multi 批量添加模式
安装 JODO

从GitHub直接安装

# 克隆仓库
git clone https://github.com/JoyinJoester/JODO.git
cd JODO
# 编译安装
cargo build --release
sudo cp ./target/release/jodo /usr/local/bin/
sudo chmod +x /usr/local/bin/jodo

使用Cargo从GitHub直接安装

cargo install --git https://github.com/JoyinJoester/JODO.git

Linux安装

1. 首先进行发布构建:

cargo build --release

2. 运行提供的安装脚本:

sudo chmod +x ./debian_install.sh
sudo ./debian_install.sh

该脚本会将编译好的二进制文件复制到 /usr/local/bin 目录，并可选地设置 PATH 环境变量。

3. 手动安装方式:

如果不想使用安装脚本，可以手动安装:

sudo cp ./target/release/jodo /usr/local/bin/
sudo chmod +x /usr/local/bin/jodo

4. 验证安装:

安装后，打开新的终端并输入:

jodo --version

如果显示版本信息，则安装成功。

使用Cargo安装

如果已经安装了Rust和Cargo，也可以使用以下命令安装:

cargo install --path .
基本用法

添加新任务

# 添加新任务 jodo "完成项目报告" # 添加带有截止日期的任务 jodo "完成项目报告" -t 2023-12-31 # 列出所有任务 jodo -l jodo
基本操作演示
$ jodo "完成项目报告"
任务 "完成项目报告" 已添加
$ jodo "准备会议材料" -t 2023-12-15
任务 "准备会议材料" 已添加 截止日期: 2023-12-15
$ jodo -l
未完成任务:
ID 描述 截止日期
-----------------------------------------
1 完成项目报告 无
2 准备会议材料 2023-12-15
命令速查表
命令 描述
jodo "任务描述" 添加新任务
jodo "任务" -t 日期 添加带截止日期任务
jodo -l 或 jodo 列出所有任务
jodo -c ID 完成任务
jodo -d ID 删除任务
jodo -h 显示帮助信息
jodo -v 显示版本信息
任务管理

编辑任务内容

jodo -e 1 "更新后的任务内容"

修改任务截止日期

jodo -e 1 -t 2023-12-25 jodo -t 1 2023-12-25 # 快捷方式

标记任务为已完成

jodo -c 1 jodo -c 1 2 3 # 一次完成多个任务

取消任务完成标记

jodo -u 1

标记任务为重要

jodo --star 1

取消任务重要标记

jodo --unstar 1

删除任务

jodo -d 1 jodo -d 1 2 3 # 一次删除多个任务

查看任务详情

jodo --show 1

批量添加模式

JODO 提供批量添加模式，方便一次添加多个任务:

# 进入批量添加模式 jodo -m
批量模式演示
$ jodo -m
批量添加模式 (输入 'exit' 退出):
jodo$> 完成第一项任务
任务 "完成第一项任务" 已添加
jodo$> 完成第二项任务
任务 "完成第二项任务" 已添加
jodo$> 学习Rust编程
任务 "学习Rust编程" 已添加
jodo$> exit
批量模式已结束，共添加 3 个任务
其他选项

显示帮助信息

jodo -h

显示版本信息

jodo -v

切换语言

jodo -L en # 英文 jodo -L zh-cn # 中文 jodo -L ja # 日语
截止日期颜色说明

JODO 使用不同颜色标记任务的截止日期状态：

  • 2023-12-01 红色加粗: 已过期
  • 2023-12-10 亮红色: 今天到期（紧急）
  • 2023-12-12 黄色: 3天内到期（即将到期）
  • 2023-12-20 正常颜色: 其他日期
数据文件

JODO 将所有任务数据保存在以下位置：

# 任务数据： ~/.jodo/tasks.json

每次对任务进行操作后，数据会自动保存到此文件。如需备份数据，只需复制该文件即可。
