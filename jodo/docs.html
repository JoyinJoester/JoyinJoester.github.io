简介

JODO 是一个轻量级命令行 Todo 应用程序，帮助您高效管理日常任务。它提供简洁的命令行界面，支持截止日期设置、重要任务标记、任务优先级排序等功能，是命令行爱好者的理想选择。

主要特性

  • 简洁的命令行界面
  • 支持任务截止日期设置
  • 重要任务标记
  • 截止日期彩色显示（逾期、紧急、即将到期）
  • 批量添加和操作任务
  • 多语言支持（中文、英文、日语）

安装

从 GitHub 直接安装

# 克隆仓库 git clone https://github.com/JoyinJoester/JODO.git cd JODO # 编译安装 cargo build --release sudo cp ./target/release/jodo /usr/local/bin/ sudo chmod +x /usr/local/bin/jodo

使用 Cargo 从 GitHub 直接安装

cargo install --git https://github.com/JoyinJoester/JODO.git

Linux 安装

  1. 首先进行发布构建：

    cargo build --release

  2. 运行提供的安装脚本：

    sudo chmod +x ./debian_install.sh sudo ./debian_install.sh

    该脚本会将编译好的二进制文件复制到 /usr/local/bin 目录，并可选地设置 PATH 环境变量。

  3. 手动安装方式：

    sudo cp ./target/release/jodo /usr/local/bin/ sudo chmod +x /usr/local/bin/jodo

  4. 验证安装：

    jodo --version

    如果显示版本信息，则安装成功。

使用 Cargo 安装

如果已经安装了 Rust 和 Cargo，也可以使用以下命令安装：

cargo install --path .

基本用法

添加任务

# 添加新任务 jodo "完成项目报告" # 添加带有截止日期的任务 jodo "完成项目报告" -t 2023-12-31

查看任务

# 列出所有任务 (两种方式都可以) jodo -l jodo
提示： 在 JODO 中，任务显示分为未完成和已完成两部分。未完成任务列在上方，已完成任务列在下方。

任务管理

编辑任务

# 编辑任务内容 jodo -e 1 "更新后的任务内容" # 修改任务截止日期 jodo -e 1 -t 2023-12-25 # 快捷方式：直接修改截止日期 jodo -t 1 2023-12-25

标记任务完成状态

# 标记任务为已完成 jodo -c 1 # 同时标记多个任务为已完成 jodo -c 1 2 3 # 取消任务完成标记 jodo -u 1c
注意： 已完成任务的 ID 会在末尾显示 'c'，例如 '1c'。使用 -u 命令取消完成标记时，需要包含这个 'c' 后缀。

标记重要任务

# 标记任务为重要 jodo --star 1 # 取消任务重要标记 jodo --unstar 1
提示： 重要任务会在列表中置顶显示，并带有星号标记。

删除任务

# 删除单个任务 jodo -d 1 # 同时删除多个任务 jodo -d 1 2 3

查看任务详情

# 显示任务的详细信息 jodo --show 1

子命令形式

JODO 也支持子命令形式进行操作：

jodo list # 列出所有任务 jodo done 1 # 标记任务为已完成 jodo undo 1c # 取消任务完成标记 jodo remove 1 # 删除任务 jodo edit 1 "新内容" # 编辑任务内容 jodo edit 1 -t 2023-12-25 # 编辑任务截止日期 jodo star 1 # 标记任务为重要 jodo unstar 1 # 取消任务重要标记 jodo show 1 # 显示任务详情

批量模式

JODO 提供批量添加任务的模式，让您可以快速连续添加多个任务：

# 进入批量添加模式 jodo -m # 示例会话: jodo$> 完成第一项任务 任务 "完成第一项任务" 已添加 jodo$> 完成第二项任务 任务 "完成第二项任务" 已添加 jodo$> 学习Rust编程 任务 "学习Rust编程" 已添加 jodo$> exit # 输入exit退出批量模式
提示： 在批量模式中，每输入一行内容并按回车，就会添加一个新任务。输入 "exit" 可以退出批量模式。

截止日期

JODO 支持为任务设置截止日期，并使用颜色来提示截止日期的紧急程度：

颜色 状态 描述
红色加粗 已过期 任务截止日期已过
亮红色 今天到期 任务在今天截止（紧急）
黄色 即将到期 任务在3天内到期
正常颜色 未紧急 任务截止日期较远

设置截止日期

# 创建任务时设置截止日期 jodo "完成项目报告" -t 2023-12-31 # 修改现有任务的截止日期 jodo -e 1 -t 2023-12-25 # 快捷方式 jodo -t 1 2023-12-25
提示： 日期格式为 YYYY-MM-DD，例如 2023-12-31。

多语言支持

JODO 支持多种语言界面，您可以使用 -L 参数切换语言：

# 切换到中文界面 jodo -L zh-cn # 切换到英文界面 jodo -L en # 切换到日语界面 jodo -L ja
提示： 语言设置会在当前会话中生效，如果您希望永久设置语言，可以在常用的shell配置文件（如 .bashrc 或 .zshrc）中添加别名。

使用技巧

任务优先级管理

  • 使用星标（--star）标记最重要的任务，它们会自动置顶
  • 对临近截止日期的任务优先处理，JODO会以颜色提示您

数据存储位置

JODO 将所有任务数据保存在以下位置：

~/.jodo/tasks.json

如果您想备份数据，只需拷贝这个文件即可。

建议的别名设置

您可以在 shell 配置文件（如 .bashrc、.zshrc 等）中添加以下别名，简化常用命令：

# 显示所有任务 alias jl='jodo -l' # 标记任务完成 alias jc='jodo -c' # 快速添加任务 alias ja='jodo' # 进入批量添加模式 alias jm='jodo -m'
