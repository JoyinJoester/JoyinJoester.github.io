下载 JODO

选择适合您操作系统的版本和安装方式

系统要求

  • 支持的操作系统: Linux
  • Rust 编译环境 (通过 rustup 安装)
  • 磁盘空间: < 10MB

快速安装

使用 Cargo 从 GitHub 一键安装最新版本:

cargo install --git https://github.com/JoyinJoester/JODO.git

安装后，运行 jodo --version 验证是否成功。

从源代码安装

克隆 GitHub 仓库并编译安装:

克隆仓库

git clone https://github.com/JoyinJoester/JODO.git cd JODO

编译项目

cargo build --release

安装到系统

sudo cp ./target/release/jodo /usr/local/bin/ sudo chmod +x /usr/local/bin/jodo

Linux 脚本安装

使用提供的安装脚本快速安装:

sudo chmod +x ./debian_install.sh sudo ./debian_install.sh

该脚本会将编译好的二进制文件复制到 /usr/local/bin 目录，并可选地设置 PATH 环境变量。

故障排除

找不到命令

如果安装后提示找不到命令，请确保安装目录在您的 PATH 环境变量中：

echo $PATH

您可能需要将以下行添加到您的 .bashrc 或 .zshrc 文件中：

export PATH=$PATH:/usr/local/bin

缺少依赖

如果编译时出现缺少依赖的错误，请确保已安装 Rust 工具链和必要的开发包：

rustup update # 在 Debian/Ubuntu 系统上 sudo apt install build-essential pkg-config

如果您仍然遇到问题，请提交 issue 到我们的 GitHub 仓库。
