📝 使用方法

AUV 提供了简单直观的命令行界面，让文件整理变得轻松

基本文件整理

auv -pdf 整理PDF文件
auv -img 整理图片文件
auv -doc 整理文档文件

自定义路径

auv -pdf here 在当前目录创建PDF文件夹
auv -img ./Pictures 移动图片到Pictures文件夹

配置管理

auv status 显示当前配置
auv set enable archive 启用压缩包文件支持
auv set path pdf ~/PDFs 设置PDF默认目标路径

历史与回退

auv history 查看操作历史
auv return 回退上一个操作
auv return T15 回退到时间线T15

🔧 自定义命令

auv -py
